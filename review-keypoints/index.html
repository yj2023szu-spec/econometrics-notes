§0全篇导论：现代计量经济学的方法谱系

★★★ 必考

脉络梳理

现代计量经济学的核心命题是"如何在非实验数据中识别因果关系"。从经典的OLS回归到最前沿的双机器学习，方法论演进遵循一条清晰的主线：

┌─→ 匹配/PSM ──→ DID ──→ 交叠DID(CSDID) │ OLS(基线) ──→ 固定效应FE ──→ 面板内生性 ──→ 2SLS/面板IV ──→ GMM(差分/系统) │ └─→ 高维方法(LASSO) ──→ Double Lasso ──→ DML/Neyman正交 │ └─→ 非线性ML(树/RF/Boosting/NN)

一句话主线：观测数据中处置分配非随机 → 需要假设识别条件 → 不同假设引出不同方法 → 方法的核心差异在于"用什么信息替代随机化"。

习题

⭐ Q0.1 简述现代计量经济学方法论的主线演进逻辑。

参考答案要点：从OLS出发，面对三大挑战——①不可观测异质性 → FE/RE；②内生性 → IV/GMM；③高维混杂 → LASSO/Double Lasso → DML。每一步都是在"更弱的假设下识别因果效应"。核心trade-off始终是"识别假设的可信度 vs 估计效率"。

★1双重固定效应 · GMM · 动态面板

★★★ 必考大题

脉络梳理

开场Puzzle：你有一份企业面板数据，研究R&D支出与规模的关系。控制了企业固定效应后系数显著——但企业规模本身可能是内生的（成功的创新→增长→规模扩大）。固定效应能解决这个问题吗？不能——FE只消除不随时间变化的遗漏变量，不能处理时变的内生性。

方法演进链：

FE(消除αi，但Xit与εit相关→不一致) → 面板2SLS(xtivreg)：对组内离差变换后的数据做2SLS → 动态面板：加入Yit-1，组内变换导致Nickell偏误(≈O(1/T)) → Anderson-Hsiao：用Yit-2作为ΔYit-1的IV → 差分GMM(Arellano-Bond)：所有可用滞后项做IV，块对角Zi矩阵 → 系统GMM(Blundell-Bond)：加入水平方程矩条件，解决γ→1时的弱IV

重点内容申明

1. 固定效应(FE)与组内变换

直觉：每个个体有自己的截距αi，吸收所有不随时间变化的特征。相当于"每个个体跟自己比"。

模型：$Y_{it} = \alpha_i + X_{it}'\beta + \varepsilon_{it}$

组内变换减去个体时间均值：$\ddot{Y}_{it} = \ddot{X}_{it}'\beta + \ddot{\varepsilon}_{it}$，αi被消除。

关键假设——严格外生性：$E(\varepsilon_{it} | X_{i1},...,X_{iT}, \alpha_i) = 0$。这比横截面OLS的外生性强得多——要求各期误差与所有时期的X都不相关。

FE的代价：时不变变量的系数无法估计（性别、种族等——被αi吸收了）。

* 固定效应模型 xtset id year xtreg y x1 x2, fe vce(cluster id) * fe选项自动做组内离差变换 * vce(cluster id) 允许个体内误差相关

2. Nickell偏误——动态面板的根源问题 ★★★ 必考

直觉：经济行为有惯性——今年的消费与去年相关。加入$Y_{it-1}$后，组内变换使$\Delta Y_{it-1}$与$\Delta\varepsilon_{it}$通过$\varepsilon_{it-1}$产生相关。

以T=3为例：差分方程$\Delta Y_{i3} = \gamma \Delta Y_{i2} + \Delta \varepsilon_{i3}$。展开：

$E(\Delta Y_{i2} \Delta \varepsilon_{i3}) = E[(Y_{i2}-Y_{i1})(\varepsilon_{i3}-\varepsilon_{i2})]$

关键项：$E(Y_{i2}\varepsilon_{i2}) = -\sigma^2_{\varepsilon} \neq 0$——这破坏了OLS的一致性。

偏误大小：≈$O(1/T)$。T=5时偏误约-20%，不可忽视。Hansen的结论：无论T多大，动态面板不能用FE。

3. Anderson-Hsiao估计量 ★★★ 核心

核心直觉：差分消除αi后，用$Y_{it-2}$作为$\Delta Y_{it-1}$的工具变量。

外生性：若εit无自相关，$Y_{it-2}$只依赖于$\varepsilon_{it-2}$及更早信息，与$\varepsilon_{it}$和$\varepsilon_{it-1}$均无关 → $E(Y_{it-2}\Delta\varepsilon_{it}) = 0$。

相关性：$Y_{it-2}$与$\Delta Y_{it-1} = Y_{it-1} - Y_{it-2}$显然相关。

* Anderson-Hsiao: 一阶差分+滞后两期做IV xtivreg y x1 x2 (L.y = L2.y), fd vce(robust) * L.y：因变量滞后一期（内生） * L2.y：因变量滞后两期（工具变量） * fd：一阶差分消除固定效应

局限：①只用一个IV，效率低；②εit必须无自相关；③γ接近1时$Y_{it-2}$变成弱IV。

4. 差分GMM (Arellano-Bond) ★★★ 必考大题

核心思想：不同时间点可用的滞后IV数量不同——t=3有1个(Yi1)、t=4有2个(Yi1,Yi2)、...。差分GMM把所有可用滞后项纳入矩条件联合估计。

工具变量矩阵Zi是块对角结构：

$$Z_i = \begin{pmatrix} [Y_{i1}, \Delta X_{i3}] & 0 & \cdots \\ 0 & [Y_{i1},Y_{i2},\Delta X_{i4}] & \cdots \\ \vdots & \vdots & \ddots \end{pmatrix}$$

矩条件：$E(Z_i'\Delta\varepsilon_i)=0$。工具变量总数$L = K(T-2) + \frac{(T-1)(T-2)}{2}$

一步GMM两步GMM
权重矩阵假设同方差先用一步残差估计S
效率较低渐近更有效
标准误普通稳健Windmeijer小样本修正

多弱工具变量问题：当T较大时IV数量$O(T^2)$膨胀。实证建议用maxldep(#)限制滞后阶数。

* 两步差分GMM（推荐） xtabond depvar indepvars, lags(2) maxldep(3) // twostep vce(robust) * lags(2)：模型中含Yit-1和Yit-2 * maxldep(3)：IV最多用到滞后3期

5. 系统GMM (Blundell-Bond) ★★★ 必考

为什么需要系统GMM？当γ→1（高度持续性）时，$Y_{it-1}$的变动几乎全部来自αi和久远冲击，$Y_{it-2}$对$\Delta Y_{it-1}$的预测能力 → 0。差分GMM的IV变成弱IV。

Blundell-Bond的洞见：为水平方程也找IV——用$\Delta Y_{it-1}$作为水平方程中$Y_{it-1}$的IV。

  • 相关性：差分与水平值显然相关
  • 外生性：若εit无自相关，$\Delta Y_{it-1}$与当期$\varepsilon_{it}$不相关
  • 附加条件（初值条件）：$E[(Y_{i1} - \frac{\alpha_i}{1-\gamma})\alpha_i] = 0$——初值偏离均值的部分与个体效应无关

系统GMM = 差分方程矩条件 + 水平方程矩条件，联合估计。蒙特卡洛模拟表明γ≈1时系统GMM小样本性质远优于差分GMM。

* xtabond2：实证首选（非官方命令） ssc install xtabond2 xtabond2 lwage L.lwage occ south smsa ind // wks L.wks ms union, // gmmstyle(lwage, lag(2 4)) // gmmstyle(wks ms union, lag(2 3)) // ivstyle(occ south smsa ind) // twostep robust * gmmstyle：GMM型IV（滞后项） * ivstyle：标准IV（严格外生变量） * 去掉noleveleq即系统GMM；加noleveleq即仅差分GMM

6. 变量分类：严格外生·前定·内生 ★★ 常考

类型定义差分后IV选取
严格外生$E(X_{it}\varepsilon_{is})=0, \forall t,s$ΔXit自身
前定$E(X_{it}\varepsilon_{is})=0, s\geq t$；与未来误差可相关Xit-1, Xit-2,...
内生$E(X_{it}\varepsilon_{it}) \neq 0$从Xit-2开始

为什么前定变量差分后内生？$\Delta X_{it}$中含$X_{it-1}$，而$X_{it}$可能与$\varepsilon_{it-1}$相关（前定变量允许与未来误差相关，但当期$X_{it}$对过去的$\varepsilon_{it-1}$...这要看具体经济含义）。关键公式：

$E(\Delta X_{it} \Delta \varepsilon_{it}) = -E(X_{it}\varepsilon_{it-1}) \neq 0$

7. 诊断检验：自相关 & 过度识别 ★★★ 必考

自相关检验(estat abond)：检验差分残差的自相关。注意：即使εit无自相关，Δεit必有一阶自相关。合格结果：一阶p<0.05，二阶(及以上)p>0.05。

过度识别检验：IV数 > 内生变量数时可检验所有矩条件是否同时成立。异方差下用Hansen检验，同方差下用Sargan检验。拒绝H0 → 某些IV不满足外生性。

* 自相关检验 estat abond, artests(3) * 过度识别检验 estat sargan // 同方差 * xtabond2自动报告Hansen检验

习题

⭐ Q1.1 Nickell偏误的来源是什么？为什么T越大偏误越小？

来源：组内离差变换使$\Delta Y_{it-1}$与$\Delta\varepsilon_{it}$通过$\varepsilon_{it-1}$产生相关（$E(Y_{it-1}\varepsilon_{it-1}) \neq 0$）。偏误≈$O(1/T)$，T增大时每个体更多期观测，组内均值的估计更精确，离差变换带来的相关性减弱。

易错点：不能说"T大时FE就一致了"——理论上偏误永远存在，只是变小。

⭐⭐ Q1.2 差分GMM和系统GMM的区别是什么？什么情况下系统GMM显著优于差分GMM？

核心区别：差分GMM仅用差分方程的矩条件；系统GMM额外加入水平方程的矩条件（用差分做IV）。

系统GMM显著优于差分GMM的场景：①γ接近1（高度持续性，如GDP、消费）；②个体效应方差远大于误差方差（$\sigma^2_\alpha/\sigma^2_\varepsilon$大）。这两种情况下差分GMM的滞后项IV变成弱IV，系统GMM的水平方程提供额外识别信息。

代价：系统GMM需要更强的假设——初值条件（均值平稳性）。

⭐⭐⭐ Q1.3 假设εit服从MA(1)过程，工具变量选取需如何调整？

若$\varepsilon_{it} = \eta_{it} + \psi\eta_{it-1}$，则$\varepsilon_{it}$与$\varepsilon_{it-1}$相关。此时$Y_{it-2}$与$\Delta\varepsilon_{it}$中的$\varepsilon_{it-1}$相关 → $Y_{it-2}$不再外生。有效IV从$Y_{it-3}$开始。Stata中调整：gmmstyle(y, lag(3 5))替换lag(2 4)。水平方程IV从$\Delta Y_{it-2}$开始。

★2随机实验 · 回归调整 · CATE · 因果推断

★★★ 核心

脉络梳理

开场Puzzle：研究"上大学对收入的因果效应"。直接比较大学毕业生和高中毕业生的平均收入——差异很大。但这是上大学的因果效应吗？不——上大学的人本身能力可能更强，即使不上大学收入也更高。这就是选择偏误(selection bias)。

随机实验(IA)：处置与潜在结果独立——黄金标准但经济数据不可得 ↓ 条件独立(CIA/CMI)：控制可观测Xi后处置近似随机 ↓ 回归调整(RA)：用参数模型"部分掉"Xi的影响 ↓ 匹配/PSM：构造可比样本，非参数地控制Xi

重点内容申明

1. 条件独立假设(CIA)与条件均值独立(CMI) ★★★ 核心

直觉：CIA = "分组后，组内随机"。虽然不控制时不可比，但在每个Xi取值相同的子群体内，处置就像抽签。

CIA：$(Y_{1i}, Y_{0i}) \perp\!\!\!\perp D_i \mid X_i$ ——要求整个分布独立。

CMI：$E(Y_{di} \mid X_i, D_i) = E(Y_{di} \mid X_i)$ ——只要求均值独立。

关键区别：CMI比CIA弱——回归只需要均值独立就够了。但CIA/CMI都不可直接检验——你必须论证"控制了足够多的变量"。

2. 同质处置效应下的回归调整 ★★★ 必考

假设$\tau_i = \tau$（所有人效应相同），在CMI+线性CEF下：

$Y_i = \alpha + \tau D_i + X_i'\gamma + \varepsilon_i$

$D_i$的系数$\tau$就是因果效应。回归相当于把Xi的影响"部分掉"(partial out)，剩下的Di与Yi的关系就是干净的因果效应。

三个条件同时成立：CMI + 同质效应 + 线性CEF → OLS给出τ的一致估计。

3. 异质处置效应与CATE ★★★ 必考

直觉：处置效应因人而异——教育回报可能因能力而异，政策效果可能因地区而异。这就是条件平均处理效应(CATE)的核心关注点。

设异质POM：

$Y_{0i} = \psi_0 + (X_i - \eta)'\gamma_0 + u_{0i}$

$Y_{1i} = \psi_1 + (X_i - \eta)'\gamma_1 + u_{1i}$

代入$Y_i = Y_{0i} + D_i(Y_{1i} - Y_{0i})$，整理得：

$Y_i = \psi_0 + \tau_{ATE} D_i + (X_i - \bar{X})'\gamma_0 + D_i(X_i - \bar{X})'\delta + \varepsilon_i$

其中$\tau_{ATE} = \psi_1 - \psi_0$，$\delta = \gamma_1 - \gamma_0$。交互项$D_i(X_i-\bar{X})$捕捉了"谁受益更多"。

CATE的实证含义：$\tau(X_i) = \tau_{ATE} + (X_i - \bar{X})'\delta$——不同特征个体的处置效应不同。

习题

⭐ Q2.1 CIA和CMI的核心区别是什么？为什么CMI对回归足够？

CIA要求$(Y_1,Y_0)$的整个联合分布独立于D|X，CMI只要求条件均值独立。回归只关心$E(Y|X,D)$，均值独立就能保证Di的系数有因果解释。CIA更强但通常不是必需的。

⭐⭐ Q2.2 异质处置效应下，为什么需要加入$D_i \times (X_i - \bar{X})$交互项？

异质时ATE不仅来自截距差还来自斜率差。从POM推导：$Y_i = Y_{0i} + D_i(Y_{1i}-Y_{0i})$，将两个POM方程代入展开，Di与$(X_i-\bar{X})$的交互项系数$\delta=\gamma_1-\gamma_0$捕捉"特征Xi如何改变处置效应"。不加交互项会把异质部分挤入误差项，导致$\hat{\tau}$不一致。

⭐⭐⭐ Q2.3 CATE（条件平均处理效应）在政策评估中的实际意义是什么？

CATE回答"谁从政策中受益更多"。ATE只给总体平均，掩盖了重要的异质性。例如：职业培训对低学历者效果更好（CATE随教育水平递减）→ 政策应精准投放。从方法角度，CATE需要交互项或分组估计，且需要更强的识别条件（CIA必须在所有X取值处成立）。参考Chernozhukov et al.的Generic ML方法可估计CATE。

★3高维数据 · ML方法 · 交叉验证 · DML

★★★ 核心难点

脉络梳理

开场Puzzle：你想评估"CEO学历对上市公司绩效的因果效应"。潜在混杂变量可能有数百个（行业、规模、年龄、治理结构、地域...）——当p接近或超过n时，OLS直接崩溃（X'X奇异）。如何在"宽数据"中可靠地估计和推断？

预测主线 推断主线 │ │ OLS(pn)→ Post-Lasso Neyman正交得分 │ │ ├──→ 岭回归(ℓ2) DML(Chernozhukov+ 2018) │ │ └──→ 非线性ML √N渐近正态推断 (RF/Boosting/NN)

一句话主线：正则化让高维估计成为可能；Neyman正交+交叉拟合让ML预测器的灵活性不损害统计推断的可靠性。

重点内容申明

1. LASSO与ℓ1正则化 ★★★ 核心

$$\hat{\beta}_{LASSO} = \arg\min_\beta \|Y - X\beta\|^2 + \lambda\|\beta\|_1$$

直觉：ℓ1惩罚的"菱形"约束域使解倾向于落在坐标轴上 → 部分系数精确为零 → 自动变量选择。这在几何上是ℓ1球与等高线的切点更容易出现在尖角处。

LASSO的核心局限：系数有收缩偏误(shrinkage bias)——即使选对了变量，$\hat{\beta}_j$也被λ压缩了。对预测这无害（甚至有益，降低了方差），但对推断有问题——不能直接用LASSO系数做置信区间和t检验。

2. Post-Lasso与选择后推断问题 ★★★

Post-Lasso = Lasso选变量 → OLS估计所选变量的系数。消除了收缩偏误，但引入新问题：

选择后推断(post-selection inference)：如果用全部数据先选变量再做OLS，标准误被系统性低估——因为变量选择步骤利用了数据中Y与X的随机相关性，这些"运气成分"在推断阶段没被考虑。

解法：样本分割——一份数据选变量，另一份做推断。Double Lasso将此思想系统化。

3. Double Lasso (BCH 2014) ★★★ 必考大题

核心问题：估计D对Y的因果效应，有大量潜在混杂Z（p可能≫n）。

三步走：

  1. Lasso Y~Z → 选出预测Y的混杂 $\hat{S}_Y$
  2. Lasso D~Z → 选出预测D的混杂 $\hat{S}_D$
  3. OLS Y~D+$\hat{S}_Y \cup \hat{S}_D$ → 对D的系数做推断

为什么取并集？单做第1步可能遗漏"对Y影响弱但对D影响强"的混杂——它们不直接预测Y但通过D间接影响结果。第2步专门捕获这类变量。取并集确保不遗漏任何重要混杂。

这是老师强调的"分步法在高维数据面前有效估计与检验假设"的核心实例。

4. 非线性ML预测器概览 ★★ 常考

方法核心思想优势劣势
回归树递归二分空间，区域常数预测完全可解释单棵方差大
随机森林Bootstrap+随机子集变量→多树平均稳健、少调参精度
XGBoost序贯种树，每棵修正前树残差精度最高需调参、易过拟合
神经网络层层非线性变换构造特征逼近任意函数需大数据、黑箱

老师特别强调的"利用神经网络捕捉非线性关系"：$h^{(l)} = \sigma(W^{(l)}h^{(l-1)} + b^{(l)})$。激活函数σ引入非线性——没有它，多层等价于单层线性变换。深度网络在图像/文本上优势明显，但在结构化表格数据上XGBoost常更优。

5. 双重机器学习(DML) ★★★ 必考大题

DML = Neyman正交得分 + 交叉拟合 + 任意ML预测器

核心思想：以部分线性模型$Y = D\theta_0 + g_0(Z) + \varepsilon$为例：

  1. 用ML估计$\hat{E}[Y|Z]$和$\hat{E}[D|Z]$
  2. 构造残差$\hat{W} = Y - \hat{E}[Y|Z]$，$\hat{V} = D - \hat{E}[D|Z]$
  3. 回归$\hat{W}$~$\hat{V}$得$\hat{\theta}$

正交化的魔力：残差化消除了$g_0(Z)$的混杂影响。得分函数对讨厌参数的导数为零 → 讨厌参数估计误差对θ的影响是二阶的 → 只要ML以$n^{-1/4}$速率收敛，θ就是$\sqrt{n}$渐近正态的。

交叉拟合：将样本分K份，轮流用K-1份估计$g_0$，在剩下1份上估计θ，最后平均。防止过拟合偏差——避免用同一份数据既训练ML又做推断。

这是老师强调的"模型选择与评估技术如交叉验证与残差分析"在因果推断中的核心应用。

6. 偏差-方差权衡与交叉验证 ★★★ 必考

$MSE = \text{Bias}^2 + \text{Variance} + \text{Irreducible Error}$

过拟合：模型太复杂 → 低偏差高方差 → 训练集完美、测试集崩溃。欠拟合：太简单 → 高偏差。

正则化通过惩罚项控制复杂度，等价于在偏差和方差之间人为设定最优平衡点。λ的选取通过K折交叉验证（常用K=5或10）。

关键警告（老师强调）：所有模型选择步骤必须在CV循环内部完成。在CV之前先做变量筛选 → 信息泄露 → CV误差被严重低估。这在实践中极其常见且危险。

7. 模型选择：AIC vs BIC ★★ 常考

$AIC = -2\log L + 2k$，$BIC = -2\log L + k\log n$

AIC目标是最小化预测误差（不假定真实模型在候选集中）；BIC目标是选出真实模型（假定它存在且n大时一致）。BIC惩罚更重（n≥8时logn>2）→ 选出模型更简洁。实证中：预测导向用AIC，因果/结构解释可用BIC参考。

习题

⭐ Q3.1 LASSO的ℓ1惩罚为什么产生稀疏解而Ridge的ℓ2惩罚不产生？

几何解释：ℓ1约束域是菱形（有尖角），ℓ2约束域是球面（光滑）。OLS等高线与ℓ1菱形更容易在坐标轴（尖角）处相切 → 某些系数精确为零。ℓ2球面处处光滑，切线几乎不可能正好在坐标轴上 → 系数被压缩但不为零。数学上，ℓ1的次梯度在零点不唯一，允许最优解落在零处。

⭐⭐ Q3.2 Double Lasso为什么必须取两个LASSO的并集而不是交集？

LASSO Y~Z选出预测Y的变量；LASSO D~Z选出预测D的变量。一个变量如果"对Y影响弱但对D影响强"，在第一个LASSO中可能被惩罚到零（因为λ的选取以预测Y为目标），但在第二个LASSO中会被选出来。如果取交集就漏掉了这类重要的混杂变量——它们通过D间接影响Y。取并集确保不遗漏任何重要混杂。

⭐⭐⭐ Q3.3 解释DML中"交叉拟合"的必要性——为什么不能在同一份数据上既训练ML又估计θ？

如果同一份数据既用于训练ML估计$\hat{g}_0$又用于估计θ，会产生过拟合偏差：ML（尤其是灵活方法如随机森林）会过度拟合Y中的随机噪声，使得残差$\hat{W}$中的人为相关性被带入第二阶段回归，导致$\hat{\theta}$有偏。交叉拟合确保每个观测值的残差来自"没见过该观测值"的ML模型——等价于样本外的残差化，切断了过拟合的传导链。Chernozhukov et al. (2018)证明K折交叉拟合下，偏误约为$O(n^{-1/2})$。

★4匹配 · 验证 · PSM · 面板固定效应 · DID

★★★ 核心

脉络梳理

开场Puzzle：评估"参加职业培训对就业的因果效应"。参加培训的人和没参加的人在年龄、学历、工作经验上本就不同。回归调整假设Y与X是线性关系——但如果不是呢？匹配方法提供了一条非参数路径：直接为每个参训者找一个"各方面相似但没参训"的对照。

精确匹配（概念清楚，高维失败）→ PSM(Rosenbaum-Rubin降维) │ └─→ DID(面板+时间差分，消除时不变不可观测混杂) │ ├─→ 2×2 DID → TWFE → 安慰剂检验 → DDD → 事件研究 │ └─→ 交叠DID(Bacon分解揭示负权重) → CSDID/Sun-Abraham

重点内容申明

1. 匹配方法的理论基础 ★★★ 核心

回归 vs 匹配：两种哲学。回归靠函数形式控制混杂——假设Y与X是线性关系。匹配靠构造可比样本——为处理组找特征相近的对照组，在"相似性空间"而非"参数空间"中做因果推断。两者都建基于CIA，当使用相同X时差异主要来自加权方式不同。

分层框架：总体ATE = 各层ATE(X)的加权平均 → $\tau_{ATE} = \sum_x ATE(x) \cdot \Pr(X_i=x)$。这给匹配提供了"分层-加权"的结构基础。

2. PSM：倾向得分的降维魔法 ★★★ 必考大题

$p(X) \equiv \Pr(D=1 \mid X)$——无论X有多少维度，p(X)始终是[0,1]上的一维标量。

两条核心性质：

  • 平衡性(Balancing)：$D \perp\!\!\!\perp X \mid p(X)$。在相同倾向得分层内，X在处理组和控制组间无系统性差异。p(X)足以"平衡"所有协变量。
  • 非混杂性(Unconfoundedness)：若CIA成立，则$(Y_1,Y_0) \perp\!\!\!\perp D \mid p(X)$。倾向得分继承了原始协变量的识别能力。

实证重心：PSM的重点不是"估计倾向得分"，而是"检验匹配后平衡性是否成立"——这正是老师强调的"匹配后还需进行事后验证，以考虑平衡性问题"。

3. PSM四步实施 ★★ 常考

  1. 估计倾向得分：Logit/Probit回归D~X
  2. 匹配：用|p(Xi)-p(Xj)|定义距离（1-1最近邻、1-M近邻、核匹配）
  3. 检验平衡性：标准化偏差SMD<0.1；不满足则回到步骤1（加交互项/平方项）
  4. 计算处置效应：$\hat{\tau}_{ATET} = \frac{1}{n_1}\sum_{i:D_i=1}(Y_i - \hat{Y}_{0i})$

两个独立要求：重合性(0<p(X)<1，支持集重叠) ≠ 平衡性(D⊥X|匹配后，分布相同)。

4. DID：双重差分 ★★★ 必考大题

核心直觉：当不可观测因素同时影响处置和结果时，CIA失效——回归调整和匹配都不可用。DID利用面板数据的时间维度，通过"差分的差分"消除不随时间变化的不可观测混杂。Card-Krueger(1994)最低工资研究是教科书案例。

$$\hat{\delta}_{DID} = [\bar{Y}_{NJ}^{post} - \bar{Y}_{NJ}^{pre}] - [\bar{Y}_{PA}^{post} - \bar{Y}_{PA}^{pre}]$$

= (政策效应+时间趋势) - (纯时间趋势) = 政策效应

平行趋势假设：无政策时处理组和控制组的时间变化应相同。不可直接检验（需要反事实），但处理前趋势可间接支持。

5. TWFE：DID的回归实现 ★★★

$Y_{it} = \alpha_i + \lambda_t + \tau D_{it} + X_{it}'\beta + \varepsilon_{it}$

αi吸收时不变差异，λt吸收共同时间冲击。τ即DID估计量。2×2时TWFE=DID；多期同处置时点时TWFE=ATT；交叠处置时TWFE可能出问题。

* 经典2×2 DID reg emp treat##post, cluster(store) * 多期TWFE xtset id year xtreg y d i.year, fe vce(cluster id) * 事件研究（平行趋势检验+动态效应） eventdd y, time(year) treated(d) cluster(id)

6. 交叠DID与Bacon分解 ★★★ 必考大题

核心问题：不同个体在不同时间接受处置(staggered adoption)时，TWFE的$\hat{\tau}$是所有可能2×2 DID的加权平均。Goodman-Bacon(2021)将其分解为三类比较：

  1. 早处理vs未处理 ✅
  2. 晚处理vs未处理 ✅
  3. 早处理vs晚处理（晚组未处置期）⚠️ 负权重！

第3类中，已处理个体被当作"控制组"——若处置效应随时间变化，这个"禁止比较"导致严重偏误，甚至可能导致符号反转。

7. 安慰剂检验与DDD ★★ 常考

时间安慰剂：政策时间前移 → "假政策"应不显著。组别安慰剂：随机打乱处置状态B次 → 真实估计应落在分布尾部。结果安慰剂：用已知不受影响的变量做Y → 应不显著。

DDD：引入第三层差分（如省份×群体），当平行趋势存疑时进一步消除特定趋势。

8. FE/RE/Hausman检验 ★★★ 必考

RE假设$\alpha_i$与$X_{it}$不相关，用GLS提高效率；FE允许相关，用组内变换消除$\alpha_i$。Hausman检验比较两者差异：H0下都一致但RE更有效→差异应小；H1下RE不一致→差异大。拒绝H0→用FE。

习题

⭐ Q4.1 PSM的平衡性和重合性有什么区别？

重合性(overlap)：0<p(X)<1——每类个体都有被处置和不被处置的可能（支持集重叠）。平衡性(balancing)：匹配后D⊥X|p(X)——两组协变量分布相同。重合性是PSM的前提条件（没有它无法匹配），平衡性是匹配质量的检验标准（匹配后必须满足）。

⭐⭐ Q4.2 Bacon分解揭示的TWFE"负权重"问题是什么？

交叠DID下，TWFE将"早处理vs晚处理（晚组未处置时）"的比较纳入估计，赋予负权重。这些比较中已处理个体被当作控制组——其Y包含处置效应的变化，污染了DID估计。当处置效应随时间异质时（如效应逐渐增大），负权重可能导致TWFE估计值方向错误。Bacon分解可以量化各类比较的权重，诊断问题严重程度。

⭐⭐⭐ Q4.3 比较Callaway-Sant'Anna和Sun-Abraham两种交叠DID估计量的异同。

相同点：①都基于分组-时间ATT(g,t)的框架；②都使用"干净"对照组（从未处理或尚未处理）；③都允许处置效应异质性；④都生成事件研究图。

不同点：CS用DR（双重稳健）估计每个ATT(g,t)，然后以队列份额为权重聚合；SA用交互加权估计量，在回归框架中估计队列特定系数后以样本份额加权。CS的DR性质使它对倾向得分或结果模型之一误设稳健；SA在控制变量设定上更灵活，可加入时变控制变量。实证建议报告两者作为稳健性检验。

★5识别条件 · 工具变量 · 双机器学习 · 因果推断

★★★ 核心

脉络梳理

开场Puzzle：你想要估计价格对需求量的因果效应。但价格由供需同时决定——直接回归得到的是供需曲线的混合，不是需求曲线。这就是经典的内生性/识别问题：没有外生变动源，就无法分离因果方向。

识别问题的根源：Y和X相互决定（联立性），或遗漏变量同时影响两者 │ ├─→ 工具变量(IV)：找到只通过X影响Y的Z——排除限制(exclusion restriction) │ └─→ 面板IV = 先组内离差再做2SLS │ ├─→ 识别条件：即使不知道真实β，也可通过允许参数微小变动来识别 │ └─→ 非线性/条件异方差提供额外识别信息（如异方差识别法） │ └─→ 双机器学习(DML)：在Neyman正交框架下，用ML估计讨厌参数 └─→ 交叉分层识别：对复杂数据结构分层处理识别问题

一句话主线：识别条件是将统计关联转化为因果解释的桥梁——IV从外部找变动源，DML从内部构造正交条件，两者的共同目标是在更弱的假设下实现可信的因果推断。

重点内容申明

1. 工具变量的识别逻辑 ★★★ 核心

2SLS的核心：第一阶段将内生变量X投影到IV空间（得到$\hat{X}$），第二阶段用$\hat{X}$替代X做回归。$\hat{X}$中只保留了X的"外生部分"——与IV相关且与误差不相关的那部分变动。

IV的两个条件：

  • 相关性：$Cov(Z, X) \neq 0$（可检验——第一阶段F>10）
  • 外生性：$Cov(Z, \varepsilon) = 0$（不可检验——必须基于经济理论论证）

面板IV：固定效应2SLS ≡ 对组内离差变换后的数据做2SLS。Stata中xtivreg自动处理变换。

* 面板固定效应2SLS xtivreg y x1 x2 (x3 = z1 z2), fe vce(robust) * x1,x2：外生协变量 * x3：内生变量 * z1,z2：排他性工具变量

2. Neyman正交得分——DML的数学核心 ★★★ 必考大题

核心定义：得分函数$\psi(W;\theta,\eta)$满足：

$$\partial_\eta \mathbb{E}[\psi(W; \theta_0, \eta)] \big|_{\eta=\eta_0} = 0$$

直觉：得分函数对讨厌参数(η)的导数为零 → 讨厌参数估计误差对目标参数(θ)的影响是二阶的 → $\hat{\eta}$不完美不影响$\hat{\theta}$的一阶渐近性质。

部分线性模型的Neyman正交得分：

$$\psi(W; \theta, \eta) = (Y - \ell(X) - \theta(D - m(X))) \cdot (D - m(X))$$

其中$\ell(X)=E[Y|X]$，$m(X)=E[D|X]$。这是"双重残差化"的得分——Y和D都先对X做残差化，本质上等价于Frisch-Waugh-Lovell定理在高维/非线性场景的推广。

3. DML的收敛速率条件 ★★★

关键定理（Chernozhukov et al., 2018）：若得分函数满足Neyman正交性，且使用K折交叉拟合，则只要ML对讨厌参数的估计速率达到$o_p(n^{-1/4})$，目标参数就是$\sqrt{n}$渐近正态的。

$n^{-1/4}$是什么概念？比$\sqrt{n}$慢得多。这意味着即使用收敛较慢的ML方法（如深度神经网络），只要收敛速率好于$n^{-1/4}$，最终推断仍然有效。这是"ML灵活性+统计推断严谨性"能够结合的关键数学保障。

直觉：Neyman正交性将ML误差的影响降为二阶 → 误差的平方进入偏误（$o(n^{-1/2}) = o((n^{-1/4})^2)$）→ ML不需要非常精确就能保证推断有效。

4. 交叉分层识别 ★★ 常考

当数据具有多层结构（如学生嵌套于班级、班级嵌套于学校），不同层级的识别条件可能不同。交叉分层识别方法允许：

  • 在同一层级内利用"类实验"变异（如班级内的随机分班）
  • 在更高层级用IV处理选择偏误（如学校的政策变化作为IV）
  • 在不同层级之间通过交互效应传递识别信息

这是老师强调的"交叉分层识别方法在处理复杂数据结构时的效用"。

5. 非线性关系与条件估计在因果识别中的应用 ★★ 常考

非线性识别来源：当函数形式为非线性时，即使没有排他性IV也可能识别因果效应。例如，Heckman选择模型依赖正态性假设识别；非参数IV利用非线性/异方差提供识别信息。

条件估计：在不同条件下（不同子群体、不同时期、不同制度环境）分别估计因果效应，然后比较异质性模式——这本身可以作为一种非正式的识别策略。如果不同条件下效应模式一致，增加因果解释的可信度。

6. 识别条件的核心地位（老师重点强调） ★★★

识别 ≠ 估计：识别是"即使有无限数据，参数能否唯一确定"的问题——是参数本身是否可从数据中分离的逻辑问题。估计是"给定有限数据，如何最佳地近似参数"——是统计效率问题。识别是估计的前提——不可识别的参数，估计再好也无意义。

老师原话的核心含义："即使面对未知的参数值，如真实贝塔值，也能通过允许参数微小变动来识别和估计因果关系"——这指向的是局部识别(local identification)的概念：参数在真实值附近的小邻域内被唯一确定。这与全局识别不同，但在很多非线性模型中已足够支撑推断。

习题

⭐ Q5.1 识别(Identification)和估计(Estimation)的区别是什么？

识别是参数是否可从观测数据的联合分布中唯一确定的逻辑问题——即使样本量无限大。估计是给定有限样本如何最优近似参数值的统计问题。不可识别的参数不能被一致估计。例如：联立方程系统中，不满足阶条件和秩条件的参数是"不可识别的"——无论样本多大，都无法区分供给曲线和需求曲线。

⭐⭐ Q5.2 解释Neyman正交得分为什么能将ML估计误差的影响降为二阶。

将估计方程$\frac{1}{n}\sum_i \psi(W_i;\hat{\theta},\hat{\eta})=0$在真实值处做Taylor展开：$\frac{1}{n}\sum_i \psi(W_i;\theta_0,\eta_0) + \frac{\partial\psi}{\partial\theta}(\hat{\theta}-\theta_0) + \frac{\partial\psi}{\partial\eta}(\hat{\eta}-\eta_0) +$ 二阶交叉项。Neyman正交性保证$\frac{\partial\psi}{\partial\eta}=0$（在期望意义上），因此$\hat{\eta}-\eta_0$的一阶项消失。剩下的误差来自$(\hat{\eta}-\eta_0)^2$量级的二阶项——只要$\hat{\eta}$以$n^{-1/4}$收敛，二阶项就是$o(n^{-1/2})$，不影响$\hat{\theta}$的$\sqrt{n}$渐近正态性。

⭐⭐⭐ Q5.3 结合DML框架，讨论"非线性关系和条件估计在识别因果效应中"的应用。

DML允许用非线性ML方法（树、森林、神经网络）估计$E[Y|Z]$和$E[D|Z]$，从而捕捉传统线性模型漏掉的非线性混杂模式。条件估计方面，DML可扩展到估计CATE：在Neyman正交框架下对每个子群体分别估计θ(x)，或使用Generic ML直接对θ(X)建模。关键创新是：非线性和异质性可以在不牺牲推断有效性的前提下被纳入——正交化+交叉拟合提供了数学保障。这正好对应老师强调的"利用神经网络捕捉非线性关系，揭示变量间复杂联系"在现代因果推断框架中的实现。

⊕全篇总结：方法决策树与速记口诀

★★★ 必考

脉络梳理

从OLS到DML，现代计量经济学提供了一套完整的方法工具箱。选择哪个方法，取决于你对数据生成过程的理解——尤其是处置分配机制和潜在混杂的性质。

五大必检项（跨方法通用）

  1. 识别假设是否可信？——CIA/平行趋势/排他性——这是所有方法的第一道防线。
  2. 诊断检验是否通过？——Hausman/abond/Hansen/平衡性/安慰剂——检验不通过就换方法。
  3. 结果对设定是否敏感？——换IV集合、换匹配参数、换控制变量——稳健性比显著性更重要。
  4. 经济意义是否合理？——系数大小、符号方向、异质性模式——统计显著≠经济显著。
  5. 结论能复现吗？——换样本时期、换变量定义、换估计方法——可靠的发现应该稳健。
