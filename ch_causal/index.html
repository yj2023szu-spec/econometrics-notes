§0 全章导论：从随机实验到观测数据

重要性：★★★

一、脉络梳理

随机实验是因果推断的黄金标准——处置与潜在结果独立(IA)。但经济数据多为被动收集，IA 不成立。如果存在一组可观测变量 Xi 使得控制后处置近似随机，则可在观测数据中识别因果效应。本章从 CIA/CMI 出发，依次介绍回归调整、匹配和倾向得分匹配。

二、重点内容申明

黄金标准与现实的差距

随机化实验（RCT）是因果推断的黄金标准。为什么？因为随机分配保证了：

$$(Y_{1i}, Y_{0i}) \perp\!\!\!\perp D_i$$

处置状态与潜在结果独立。处理组和控制组在一切特征上都可比——包括可观测和不可观测的。组间结果差异就是因果效应。

但经济学数据几乎都不是实验数据。 企业是否上市、个人是否上大学、地区是否推行某项政策——这些决策都不是随机的。处理组和控制组在"原材料"上就不同。直接比较组间均值，你不知道差异来自处置还是来自初始禀赋。

解决的钥匙：如果存在一组可观测变量 $X_i$，使得控制它们之后，处置分配近似随机：

$$(Y_{1i}, Y_{0i}) \perp\!\!\!\perp D_i \mid X_i \tag{1}$$

这就叫条件独立性假设 (CIA)。它的直觉是：虽然不控制时不可比，但在每个 $X_i$ 取值相同的子群体内，处置就像抽签。

本章的方法谱系：

随机实验(IA) ──不可得──→ 条件独立(CIA) ──更强──→ 条件均值独立(CMI)
                              │                        │
                              ▼                        ▼
                         匹配/PSM                 回归调整(RA)

三问自检：①为什么 RCT 是黄金标准？随机分配保证处置与潜在结果独立。②CIA 比 IA 弱在哪？需要控制可观测变量，但不保证不可观测因素平衡。③CIA 不可检验——你只能论证"控制了足够多的变量"。

三、习题

  1. ⭐ 随机实验为什么是因果推断的黄金标准？
参考答案要点随机分配保证 (Y1,Y0)⊥D，处理组和控制组在所有特征上可比，组间结果差异=因果效应。不需要控制任何协变量。
  1. ⭐⭐ 观测数据中 CIA 为什么"不可检验"？研究者通常如何论证其可信度？
参考答案要点CIA 涉及反事实结果——你永远看不到同一个体的 Y1 和 Y0。论证策略：①说明控制了影响处置和结果的主要可观测因素；②进行安慰剂检验（用已知不应有处置效应的结果变量）；③讨论可能遗漏的混杂变量及其方向。

§1 条件独立与条件均值独立

重要性：★★★ 核心

一、脉络梳理

CIA 要求整个分布独立，CMI 只要求均值独立——后者是回归调整的充分条件。核心直觉：固定 Xi 后，处置分配就像随机的。

二、重点内容申明

CIA 的直观理解

条件独立性假设 $(Y_1, Y_0) \perp\!\!\!\perp D \mid X$ 的核心含义是：在固定 $X$ 后，处置分配就像随机的。

以"是否选修新财经课程对毕业后收入的影响"为例：

  • 不同专业、性别的学生选修倾向不同
  • 这些因素也影响毕业收入
  • 不控制时，选课与收入的关系混杂了专业和性别的效应
  • 但如果在同一专业、同一性别的组内，选课系统对超额报名学生随机踢人
  • 那么在每个 $(专业, 性别)$ 组内，是否选课近似随机

这就是 CIA 的直觉：分组后，组内随机。

CMI：回归需要的更弱条件

CIA 要求整个分布独立。但多元线性回归其实只需要条件均值独立 (CMI)：

$$E(Y_{1i} \mid X_i, D_i) = E(Y_{1i} \mid X_i) \tag{2}$$ $$E(Y_{0i} \mid X_i, D_i) = E(Y_{0i} \mid X_i) \tag{3}$$

含义：控制了 $X_i$ 之后，潜在结果的条件均值不再依赖处置状态。CMI 比 CIA 弱——它只要求"均值独立"而非"整个分布独立"。对回归来说，均值独立就够了。

三问自检：①CIA 和 CMI 的核心区别？CIA 要求分布独立，CMI 只要求均值独立。②为什么 CMI 对回归足够？回归只关心条件期望 E(Y|X,D)，均值独立就能保证系数有因果解释。③CIA 或 CMI 可检验吗？不可直接检验——你需要看到反事实结果。

三、习题

  1. ⭐ CIA 和 CMI 的核心区别是什么？
参考答案要点CIA 要求整个条件分布独立：(Y1,Y0)⊥D|X。CMI 只要求条件均值独立：E(Yd|X,D)=E(Yd|X)。CMI 比 CIA 更弱，对回归调整而言 CMI 已足够——因为回归只关心条件期望。
  1. ⭐⭐ 如果用 CIA 而非 CMI 做回归调整，会不会有问题？
参考答案要点不会。CIA⇒CMI，更强的假设不会破坏一致性。但用更弱的 CMI 做论证更诚实——你只需要承诺"均值独立"而非"整个分布独立"。

§2 同质处置效应下的回归调整

重要性：★★★ 必考

一、脉络梳理

假设每个个体的处置效应相同 τ，在 CMI 下 Yi 对 (Di, Xi) 回归，Di 的系数即 τ。这是最简单也最直观的回归因果推断框架。

二、重点内容申明

最简单的因果回归

假设处置效应对所有人都相同：$Y_{1i} - Y_{0i} = \tau$。由潜在结果模型：

$$Y_i = Y_{0i} + \tau D_i$$

取条件期望，在 CMI 下 $E(Y_{0i} \mid X_i, D_i) = E(Y_{0i} \mid X_i)$：

$$E(Y_i \mid X_i, D_i) = E(Y_{0i} \mid X_i) + \tau D_i$$

如果进一步假设 $E(Y_{0i} \mid X_i) = \alpha + X_i'\gamma$（线性 CEF），则：

$$Y_i = \alpha + \tau D_i + X_i'\gamma + \varepsilon_i \tag{4}$$

这就是回归调整模型。 $\tau$ 就是因果效应——处置变量 $D_i$ 的系数。在 CMI + 同质效应 + 线性 CEF 三个条件下，OLS 给出 $\tau$ 的一致估计。

直觉：回归调整相当于把 $X_i$ 的影响"部分掉"（partial out），剩下的 $D_i$ 与 $Y_i$ 的关系就是干净的因果效应。

三问自检：①同质效应的限制性？假设每个人对处置反应相同，实际上不太可能。②$\gamma$ 有因果解释吗？没有——$X_i$ 可能只控制混杂而非因果。③什么时候 OLS=因果？CMI+同质效应+线性 CEF 同时成立时。

三、习题

  1. ⭐ 写出同质 RA 的回归方程，并解释每个系数的含义。
参考答案要点Yi=α+τDi+Xi'γ+εi。α：当 Xi=0 时控制组的平均结果；τ：因果效应（Di 变化 1 单位对 Yi 的影响）；γ：Xi 与 Y0i 的关联（不是因果效应）。
  1. ⭐⭐ 同质 RA 下，OLS 给出 τ 的一致估计需要哪些假设？
参考答案要点①CMI：E(Y0|X,D)=E(Y0|X)；②同质处置效应：Y1i-Y0i=τ 对所有 i 相同；③线性 CEF：E(Y0|X)=α+X'γ。三个条件缺一不可——任何一个违反都可能使 τ̂ 不一致。

§3 异质处置效应下的回归调整

重要性：★★★

一、脉络梳理

处置效应因人而异时，回归需加入 Di×(Xi-X̄) 交互项。ATE 由截距差 ψ1-ψ0 给出。推导关键：将 POM 代入 Y = Y0 + D(Y1-Y0)。

二、重点内容申明

处置效应因人而异

现实中的处置效应通常异质。设潜在结果的线性 CEF 为：

$$Y_{0i} = \psi_0 + (X_i - \eta)'\gamma_0 + u_{0i} \tag{5}$$ $$Y_{1i} = \psi_1 + (X_i - \eta)'\gamma_1 + u_{1i} \tag{6}$$

其中 $\eta = E(X_i)$（中心化协变量）。个体处置效应：

$$\tau_i = Y_{1i} - Y_{0i} = [\psi_1 - \psi_0] + (X_i - \eta)'(\gamma_1 - \gamma_0) + [u_{1i} - u_{0i}]$$

取期望，中心化项的期望为零：

$$\tau_{ATE} = E(\tau_i) = \psi_1 - \psi_0$$

ATE 由潜在结果方程的截距差给出。 这很简洁。

回归实现

将 POM $Y_i = Y_{0i} + D_i(Y_{1i} - Y_{0i})$ 代入整理：

$$Y_i = \psi_0 + \tau_{ATE} D_i + (X_i - \eta)'\gamma_0 + D_i(X_i - \eta)'\delta + \varepsilon_i$$

其中 $\delta = \gamma_1 - \gamma_0$。在 CMI 下，$\varepsilon_i$ 的条件期望为零。将 $Y_i$ 对 $(D_i, X_i, D_i(X_i - \bar{X}))$ 回归，$D_i$ 的系数即 $\tau_{ATE}$。

三问自检：①为什么加入交互项？处置效应异质时，需要 Di×(Xi-X̄) 来捕捉"谁受益更多"。②δ 的符号含义？δ>0 说明高 Xi 者处置效应更大。③同质是异质的特例？是的，γ1=γ0 时 δ=0，退化到 §2。

三、习题

  1. ⭐ 异质 RA 中为什么需要加入 Di×(Xi-X̄) 交互项？
参考答案要点当处置效应因人而异时，Di 的系数不再直接等于 ATE——它混淆了"处置的平均效应"和"处置效应与 Xi 的相关性"。交互项将异质部分分离出来，使得 Di 的系数恰好捕捉 ATE。
  1. ⭐⭐ 推导：从 Y0i 和 Y1i 的线性 CEF 出发，证明异质 RA 中 τATE=ψ1-ψ0。
参考答案要点τi=Y1i-Y0i=[ψ1-ψ0]+(Xi-η)'(γ1-γ0)+[u1i-u0i]。取期望：E(τi)=ψ1-ψ0+E(Xi-η)'(γ1-γ0)+E(u1i-u0i)。因为 E(Xi-η)=0（中心化），E(u1i)=E(u0i)=0（CEF 误差均值为 0），所以 τATE=ψ1-ψ0。

§5 匹配方法的基本思想

重要性：★★★ 核心

一、脉络梳理

回归强调函数形式，匹配强调构造可比样本。核心：为处理组找特征相似的控制组，组间差异归因于处置。理论基础是分层——同一层内ATE(X) 用均值差估计。

二、重点内容申明

回归 vs 匹配：两种哲学

回归调整强调函数形式——你假设 Y 与 X 是线性关系，然后用参数模型控制混杂。匹配强调构造可比样本——你直接为每个处理组个体找一个特征相似的控制组个体，把苹果跟苹果比。

两者都建立在"依可观测变量选择"基础上。当它们使用相同 X 时，区别主要来自加权方式不同。

匹配的理论基础：分层

匹配可理解为按 $X$ 对样本分层。同一层内 $X$ 相同或相近。层内平均处置效应：

$$ATE(X) = E(Y_1 - Y_0 \mid X)$$

在 CMI 下：

$$E(Y \mid X, D=1) - E(Y \mid X, D=0) = E(Y_1 \mid X) - E(Y_0 \mid X) = ATE(X)$$

由全期望公式：

$$\tau_{ATE} = E[ATE(X)] = \sum_x ATE(x) \cdot \Pr(X_i = x) \tag{7}$$

关键洞察：总体 ATE 是各层 ATE 的加权平均。这给了匹配一个清晰的"分层-加权"结构。

三问自检：①匹配和回归的核心区别？回归靠函数形式控制混杂，匹配靠构造可比样本来控制。②为什么分层后可以估计因果效应？CMI 保证层内处置近似随机。③$X_M$ 和 $X_{RA}$ 有什么不同？匹配中主要挑影响处置的变量，回归中主要挑影响结果的变量。

三、习题

  1. ⭐ 匹配和回归的核心区别是什么？
参考答案要点回归靠函数形式假设（线性 CEF）控制混杂；匹配靠构造可比样本来控制混杂。回归说"假设 Y 与 X 是线性关系然后估计"，匹配说"找到 X 相似的人然后直接比较"。
  1. ⭐⭐ 证明：CMI 下，层内结果均值差等于 ATE(X)。
参考答案要点E(Y|X,D=1)-E(Y|X,D=0)=E(Y1|X,D=1)-E(Y0|X,D=0)。由 CMI：=E(Y1|X)-E(Y0|X)=ATE(X)。CMI 允许我们将条件均值中的 D 条件去掉。

§6 精确匹配与维度诅咒

重要性：★★

一、脉络梳理

精确匹配概念最清楚但最不实用——高维/连续协变量下几乎必然失败。这是 PSM 的动机。

二、重点内容申明

精确匹配：最自然但最不可行

当 $X$ 只含离散变量时，精确匹配最直接：只在控制组中找 $X_i$ 完全相同的样本。层内均值差 = ATE(x)。

但问题来了：

  1. 维度诅咒：$X$ 含 K 个二值变量 → 2^K 个层。K=20 就有百万层，远超样本量
  2. 空层问题：某些层只有处理组或只有控制组 → 该层 ATE 无法计算
  3. 连续变量：$X$ 含连续变量时，几乎不可能有完全相同的两个观测值

精确匹配概念最清楚，但在高维或连续协变量场景下几乎必然失败。这正是 PSM 出现的动机。

三、习题

  1. ⭐ 什么是维度诅咒？举例说明。
参考答案要点协变量维度增加时，精确匹配所需的层数指数增长。例：10 个二值变量→1024 层；20 个→百万层。样本量远不足以填满所有层，导致大量"空层"无法估计 ATE(X)。
  1. ⭐⭐ 为什么维度诅咒催生了 PSM？
参考答案要点精确匹配需要 X 完全相同→高维几乎不可能。PSM 用一维的 p(X)=Pr(D=1|X) 替代多维 X 进行匹配，既保留了协变量的"混杂信息"，又避免了维度爆炸。Rosenbaum-Rubin(1983) 的两条性质保证了这种降维不损失识别能力。

§7 倾向得分匹配（PSM）

重要性：★★★ 必考大题

一、脉络梳理

倾向得分 p(X)=Pr(D=1|X) 将任意维度 X 压缩为一维。两条核心性质：平衡性 D⊥X|p(X)，非混杂性 (Y1,Y0)⊥D|p(X)（CIA 下）。PSM 的重点是检验平衡性。

二、重点内容申明

降维的魔法

倾向得分定义为：

$$p(X) \equiv \Pr(D=1 \mid X) = E(D \mid X) \tag{8}$$

它是给定 $X$ 后个体被处置的概率。无论 $X$ 有多少维度，$p(X)$ 始终是一维标量。

两条核心性质

(a) 平衡性 (Balancing)

$$D \perp\!\!\!\perp X \mid p(X)$$

在相同倾向得分层内，$X$ 的分布在处理组和控制组之间没有系统性差异。换句话说，$p(X)$ 足以"平衡"所有协变量。

(b) 非混杂性 (Unconfoundedness)

若 CIA 成立，则：

$$(Y_1, Y_0) \perp\!\!\!\perp D \mid p(X)$$

倾向得分继承了原始协变量的识别能力。用 $|p(X_i) - p(X_j)|$ 替代原始协变量距离做匹配，同样可以恢复因果效应。

实证含义

PSM 的重点不仅是"估计倾向得分"，更是"检验匹配后的平衡性是否成立"。 如果平衡性不满足，回到 Logit/Probit 模型重新设定，加交互项或高阶项，直到通过。

三问自检：①为什么 p(X) 能降维？p(X)=Pr(D=1|X) 是将多维 X 映射到 [0,1] 的函数。②平衡性和非混杂性的区别？平衡性说的是 X 与 D 的关系，非混杂性说的是 Y 与 D 的关系。③什么情况下 PSM 无效？不满足 CIA（有不可观测混杂）或重合性失败（某类个体必然被处置）。

三、习题

  1. ⭐ 写出倾向得分的定义，解释为什么它是"降维魔法"。
参考答案要点p(X)=Pr(D=1|X)=E(D|X)。无论 X 有多少维度，p(X) 始终是一维标量，取值范围 [0,1]。用 |p(Xi)-p(Xj)| 替代原始 Mahalanobis 距离匹配，大大简化了高维匹配问题。
  1. ⭐ PSM 的两条核心性质是什么？
参考答案要点(a) 平衡性 D⊥X|p(X)：相同倾向得分层内，X 在两组间无系统差异。(b) 非混杂性 (Y1,Y0)⊥D|p(X)（CIA 成立时）：倾向得分足以恢复因果识别。
  1. ⭐⭐ 为什么 PSM 的实证重点不是"估计得分"而是"检验平衡性"？
参考答案要点倾向得分的估计本身不保证因果识别——Logit 总是能估计出一个 p(X)。关键是：匹配后 X 在两组间是否真的平衡了？如果不平衡，说明降维过程中丢失了信息，需重新设定倾向得分模型（加入交互项、平方项等）。
  1. ⭐⭐ 平衡性和非混杂性的关系是什么？CIA 在哪一步起作用？
参考答案要点平衡性是 X 的统计性质：D⊥X|p(X)——不依赖任何因果假设。非混杂性依赖 CIA：(Y1,Y0)⊥D|X⇒(Y1,Y0)⊥D|p(X)。CIA 在"从 X 到 p(X)"这一步起作用——如果 CIA 不成立，即使倾向得分平衡性完美也不够。

§8 PSM 实施与诊断

重要性：★★★

一、脉络梳理

四步：估计得分→匹配→检验平衡性→计算 ATE/ATET/ATENT。重合性(0

二、重点内容申明

四步实施流程

  1. 估计倾向得分：Logit/Probit 回归 $D_i$ 对 $X_i$
  2. 匹配：用 $|p(X_i) - p(X_j)|$ 定义距离，执行最近邻/核匹配
  3. 检验平衡性：不满足则回到第 1 步重设模型
  4. 计算处置效应：

$$\hat{\tau}_{ATET} = \frac{1}{n_1}\sum_{i:D_i=1}(Y_i - \hat{Y}_{0i}) \tag{9}$$ $$\hat{\tau}_{ATENT} = \frac{1}{n_0}\sum_{i:D_i=0}(\hat{Y}_{1i} - Y_i) \tag{10}$$ $$\hat{\tau}_{ATE} = \frac{n_1}{n}\hat{\tau}_{ATET} + \frac{n_0}{n}\hat{\tau}_{ATENT} \tag{11}$$

两个独立要求

三、习题

  1. ⭐ 写出 PSM 的四步实施流程。
参考答案要点①Logit/Probit 估计 p(X)；②用 |p(Xi)-p(Xj)| 匹配；③检验平衡性（SMD<0.1）；④计算 ATE/ATET/ATENT。不通过第③步则回到第①步重新设定模型。
  1. ⭐ 重合性和平衡性的区别是什么？
参考答案要点重合性：0
  1. ⭐⭐ 1-1 最近邻匹配和核匹配的优劣势比较。
参考答案要点1-1：只用最近的一个控制组→偏差可能较小但方差大（只用很少的信息）。核匹配：用所有控制组加权→方差小但可能引入不太相似的个体（偏差可能增大）。通常建议用 1-M（M=3~5）或卡尺内匹配作为折中。
  1. ⭐⭐ 如果重合性检验失败（部分区域 p(X)=0 或 1），如何处理？
参考答案要点①Trimming：去掉倾向得分极端值（p<0.05 或 p>0.95）的观测值；②重新定义处置组/控制组：可能有些个体实际上"不可能/必然被处置"；③注意：Trimming 改变了估计的目标人群——从 ATE 变为"有重叠区域人群的 ATE"。

§9 全章总结与综合习题

重要性：★★★

一、脉络梳理

方法选择决策：有随机实验→IA；观测数据+CIA→RA 或匹配；高维X→PSM。全章速记口诀。

二、重点内容申明

方法选择决策树

处置分配机制？
  ├── 随机实验 → IA 成立，直接比较均值
  └── 观测数据
      ├── 低维离散 X → 精确匹配 + 分层
      └── 高维/连续 X
          ├── 线性 CEF 可接受 → 回归调整(RA)
          └── 更关注可比样本 → PSM
              → 1. Logit 估计 p(X)
              → 2. 匹配 + 平衡性检验
              → 3. 计算 ATE/ATET/ATENT

速记口诀

RCT 金标准，IA 最简单。观测数据难，CIA 来帮忙。回归靠参数，匹配看距离。高维有诅咒，PSM 降维破。平衡性必检，重合性要看。ATT 处理组，ATE 全样本。

五个必检项

  1. CIA/CMI 是否可信（是否有遗漏的混杂变量？）
  2. 重合性是否满足（倾向得分分布是否有重叠？）
  3. 平衡性是否通过（SMD 是否小于 0.1？）
  4. 结果对匹配参数是否敏感（M 值、卡尺 caliper 等）
  5. 是否有不可观测混杂的稳健性讨论

三、习题

  1. ⭐⭐ 画出自随机实验到 PSM 的方法选择决策树。
参考答案要点参见内容申明 §9。关键分支：随机实验→IA；观测数据→CIA/CMI；低维离散→精确匹配；高维→RA 或 PSM。每个节点标注关键假设和诊断检验。
  1. ⭐⭐ 一篇论文使用 PSM 但没有报告平衡性检验结果，审稿人可能提出什么质疑？
参考答案要点①匹配是否真的改善了平衡性？没有 SMD 比较无法判断；②PSM 的第一步 Logit 设定是否正确？是否尝试过不同的函数形式？③结论是否对匹配参数（M、caliper）敏感？建议：报告匹配前后各协变量的 SMD，展示倾向得分分布图，做 Rosenbaum 敏感性分析。
  1. ⭐⭐⭐ 查阅 Rosenbaum-Rubin (1983) 原文，简述平衡性性质的证明思路。
参考答案要点R&R 的核心论证：Pr(D=1|X,p(X))=Pr(D=1|p(X))——给定倾向得分后，X 不再提供关于 D 的额外信息。因此，在 p(X) 的每个取值层内，D 与 X 独立。证明关键：利用 Pr(D=1|X)=p(X) 是 X 的函数，迭代期望律 E(D|p(X))=p(X)。
